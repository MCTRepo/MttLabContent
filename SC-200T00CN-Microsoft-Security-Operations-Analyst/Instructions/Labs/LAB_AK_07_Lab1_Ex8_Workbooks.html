模块 7 - 实验室 1 - 练习 8 - 创建工作簿

实验室场景

你是一位安全运营分析师，你所在公司已实现 Azure Sentinel。你需要设计带有高级可视化的工作簿。

任务 1：探索工作簿。

在此任务中，你将探索工作簿的配置。

  1. 使用以下密码以管理员身份登录到 WIN1 虚拟机： Pa55w.rd。

  2. 在 Microsoft Edge 浏览器中，导航到 Azure 门户 (https://portal.azure.com)。

  3. 在“登录”对话框中，复制粘贴实验室托管提供者提供的租户电子邮件帐户，然后选择“下一步”。

  4. 在“输入密码”对话框中，复制粘贴实验室托管提供者提供的租户密码，然后选择“登录”。

  5. 在 Azure 门户的搜索栏中，键入“Sentinel”，然后选择“Azure Sentinel”。

  6. 选择 Azure Sentinel 工作区。

  7. 选择“工作簿”。

  8. 选择“标识和访问”，然后选择“查看模板”按钮。

  9. 查看“标识和访问”工作簿。

  10. 选择右上角的“x”关闭工作簿。现在，你应位于“**Azure Sentinel。 工作簿 模板**”选项卡。

  11. 在“模板”选项卡中搜索并选择“Azure AD 审核日志”，然后从右侧窗格中选择“保存”按钮。

  12. 选择你所在位置对应的位置，然后选择“确定”。适当的位置通常是默认位置。

  13. 选择“查看保存的工作簿”按钮。

  14. 在命令栏中选择“编辑”。

  15. 在“用户活动”区域中，查看图表中“操作计数”和“趋势”的颜色。然后选择网格下面的“编辑”。

若要设置列格式，可使用“列设置”面板中的自定义选项执行以下操作：

  1. 选择“查询命令栏”右侧的“列设置”按钮。

备注： 此按钮仅在报表中有数据时才会出现。

  1. 在“编辑列设置”边栏选项卡中，选择“操作计数(热图 + 格式化)”列。

  2. 检查设置，特别是列呈现器对应的选项。对于颜色调色板，选择“32 色分类”。

  3. 选择“趋势(火花线图 +格式化)”列。

  4. 检查设置，对于列呈现器，选择“火花区域”，并在“调色板”中选择所选的颜色。

  5. 选择“保存并关闭”。

若要让某个磁贴/网格控件筛选另一个磁贴/网格中的结果，请执行以下操作：

  1. 在“编辑查询项: 查询 - 2”中选择“高级设置”按钮。

  2. 查看“选择这些项目时，请导出参数”设置。注意已选择“UserInfo”字段。

  3. 在查询底部选择“完成编辑”。

  4. 在屏幕右侧的“最活跃用户”网格下选择“编辑”。

  5. 在查询中找到“UserInfo”。查询将使用从另一查询导出的参数来筛选结果。

  6. 在查询底部选择“完成编辑”。

  7. 在工作簿的顶部菜单中选择“完成编辑”，然后选择“保存”图标。选择右上角的“x”关闭工作簿。

任务 2：创建工作簿。

在此任务中，你将新建一个带有高级可视化的工作簿。

  1. 返回到 Azure Sentinel 门户中的“工作簿”区域。

  2. 选择“+ 添加工作薄”

  3. 选择“编辑”

编辑标题文本：

  1. 选择工作簿标题下方的“编辑”，将“## 新工作簿”更改为“## 我的工作簿”。

  2. 对于标题文本，选择“完成编辑”。

  3. 选择唯一可见图形下面的“编辑”。

  4. 查看可提供多个表的联合计数的 KQL 语句。

  5. 对于图形，选择“完成编辑”。

  6. 选择图形区域旁的“…”，然后选择“+ 添加”，再选择“添加查询”。

  7. 在查询框中输入“SecurityEvent”，然后选择“运行查询”按钮。

  8. 将时间范围更改为“过去 3 天”。

  9. 将“可视化效果”更改为条形图并查看结果。

  10. 将“可视化效果”更改为时间图并查看结果。

  11. 选择“查询”选项卡中的“样式”。

  12. 选择“将此项设置为自定义宽度”框。

  13. 将宽度百分数设置为 75，将最大宽度设置为 75。

  14. 选择“查询”选项卡中的“高级设置”。

  15. 选择“未进行编辑时显示刷新图标”框。

  16. 对于新图形，选择“完成编辑”。

  17. 在工作簿的底部选择“添加”，然后选择“添加查询”。

输入以下 KQL 命令进行查询：

code
SecurityEvent

  1. 对于“时间范围”，选择“过去 4 小时”。

  2. 将可视化效果更改为“网格”。

  3. 选择“样式”选项卡。

  4. 选择“将此项设置为自定义宽度”。

  5. 将百分数宽度更改为 25，将最大宽度更改为 25。

  6. 对于新查询，选择“完成编辑”。

  7. 对于“工作簿”，选择“完成编辑”。

  8. 选择“保存”图标，将“标题”更改为“我的工作簿”，然后再次选择“保存”。

  9. 选择右上角的“x”关闭工作簿，或者在 Azure Sentinel 门户中选择“工作簿”。

  10. 如果需要，请选择“我的工作簿”选项卡。

  11. 选择刚创建的工作簿。

  12. 选择“查看保存的工作簿”。

你已完成本实验室。
