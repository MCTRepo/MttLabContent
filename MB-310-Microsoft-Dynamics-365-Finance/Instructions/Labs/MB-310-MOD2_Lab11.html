Exercise 11: Create a bank group and bank account

USMF has opened a new bank account. The bank account will operate primarily in British pound (GBP) but may operate in other currencies. Annie, the bookkeeper, must create a new bank group and new bank account in Dynamics 365 Finance for this bank. Annie does not have address information for the bank account, but she does have the following specifications:

For the bank group:

  • The bank group name is RBUK for Royal Bank of UK.

  • The bank group routing number is 8117.

For the bank account:

  • The bank account name is Bank Account - GBP.

  • The bank account is part of the Royal Bank of UK group.

  • The bank account number is 57252193.

  • Postings should use a new main account 110150 called Bank account - GBP

As a finance consultant follow the instructions below to help Annie.

Create a new account for Cash in bank - GBP to the Chart of accounts.

  1. Navigate to General ledger > Chart of accounts > Accounts > Main accounts.

  2. Verify to see if you have the account 110150 – Bank Account GBP. If it exists skip to step 9.

  3. Select the New button to create a new record.

  4. In the Main account field, enter 110150.

  5. In the Name field, enter Bank Account – GBP.

  6. In the Main account type field, select Asset.

  7. In the Default Currency field, and then select GBP.

  8. Close the form

Create the new bank group and assign the British pound (GBP) to it.

  1. Navigate to Cash and bank management > Setup > Bank groups.

  2. Select the New button to create a new record.

  3. In the Bank groups field, enter RBUK.

  4. In the Routing number field, type 200395.

  5. In the Description field, enter Royal Bank of UK.

  6. Select the General FastTab.

  7. From the Currency list select GBP.

  8. Close the Bank groups form.

Create the new bank account that is part of the RBUK bank group.

  1. Navigate to Cash and bank management > Bank accounts > Bank accounts.

  2. Select the New button to create a new record.

  3. In the Bank account field, enter GBPBANK.

  4. In the Name field, enter Bank Account - GBP.

  5. In the Bank groups dropdown, select RBUK.

  6. The Routing number field automatically will display 200395.

  7. The Currency field automatically displays GBP, which defaults from the bank group.

  8. In the Bank account number field, enter 57252193.

  9. In the Main account field, select 110150.

  10. In the Allow transactions in additional currencies field, select the Yes to enable posting in more than one currency for this bank account.

  11. Close the form

Create a direct debit mandate for a customer

  1. Navigate to Accounts receivable > Customers > All customers.

  2. Select a customer, for example, select US-001.

  3. On the Action Pane, select Customer.

  4. Select Bank accounts in the Setup group.

  5. Select New

  6. In the Bank account field, type a value such as CustBank1.

  7. In the Name field, type a value such as Customer Bank 1.

  8. In the Bank account number field, enter 55555555.

  9. In the Routing number field, enter 202015.

  10. I the IBAN field, type GB33BUKB20201555555555.

  11. In the Currency field, type GBP.

  12. Select Save

  13. Close all pages

  14. Navigate to Cash and bank management > Bank accounts > Bank accounts.

  15. In the list, find and select GBPBANK.

  16. Select Edit

  17. Expand the Additional identification section.

  18. In the Direct debit ID field, type a value of US001.

  19. In the IBAN field, type GB33BARC20039557252193.

  20. Close all pages

Define the electronic payment method

  1. Navigate to Accounts receivable > Payments setup > Methods of payment.

  2. Select New

  3. In the Method of payment field, type a value such as AREL.

  4. In the Description field, type a value such as AR Electronic payment.

  5. In the Payment type field, select Electronic payment. Note that payment type for a direct debit mandate method of payment must be Electronic payment.

  6. In the Require mandate field, select Yes.

  7. Close the page

Add a direct debit mandate to a customer

  1. Navigate to Accounts receivable > Customers > All customers.

  2. Select customer US-001

  3. Select Edit

  4. Expand the Payment defaults section.

  5. In the Method of payment field, enter or select AREL.

  6. Expand the Direct debit mandates section.

  7. Select Add

  8. In the Bank account field, note that CustBank1 is automatically selected.

  9. In the Creditor bank account field, select GBPBANK.

  10. Enter the number of payments that you expect to process for this mandate, such as 10, in Expected number of payments.

  11. Select OK

  12. Note that the value of Mandate status column is set to incomplete.

  13. Select Print in the Direct debit mandates section and select Mandate report.

  14. Review the SEPA Direct Debit Mandate report.

  15. Close the page

  16. Select Edit

  17. In the Signature date field, enter a date in future.

  18. Select Yes

  19. Enter the location where the mandate was signed. Select London.

  20. Select OK

  21. Note the value of Mandate status column is set to New.

  22. Close the page
