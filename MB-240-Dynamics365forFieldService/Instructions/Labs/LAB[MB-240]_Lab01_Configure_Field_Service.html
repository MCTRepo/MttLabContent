Practice Lab 1 - Configure Dynamics 365 Field Service

Scenario

Worldwide Industries (WWI) provides IT and networking services to their customers. Their services range from phone system and network installations to telephoning systems and security system installations. They are going to be leveraging Dynamics 365 for Field Service for installation and servicing of these systems for their customers. You are the system implementor that has been tasked with configuring the application to support the rollout of the application. You will be adding and configuring some products that can be installed and setting up skills and characteristics that will be used as part of the implementation.

Task 1 - Enable Bing Maps to use with Resource Scheduling

To ensure that you are able to take full advantage of the full scheduling and mapping capabilities available with Field Service, you need to ensure that it is configured to use a mapping provider. Bing Maps is the default map provider, but additional providers could be enabled.  We will be using Bing Maps.

  1. In the Dynamics 365 Field Service app, click the Resources area in the bottom-left of the sitemap, and in the Administration group select Scheduling Parameters.

  2. Open the Resource Scheduling record.

  3. Locate the Connect to Maps field. This should be set to Yes.

  4. If Connect to Maps is set to No, set it to Yes and click OK from the popup.

  5. Click Save and Close.

Exercise 2 - Configure Territories

In this exercise, you will adding territories that will be used when scheduling resources and work orders.

Task 1 - Define Territories

  1. In the Dynamics 365 Field Service app, click the Settings area in the bottom-left of the sitemap, and in the General group select Territories.

  2. Click + New located on the command bar.

  3. Enter [your prefix ex. mollyc] + North for Territory Name and click Save. Click + New.

  4. Enter [your prefix ex. mollyc] + South for Territory Name and click Save. Click + New.

  5. Enter [your prefix ex. mollyc] + East for Territory Name and click Save. Click + New.

  6. Enter [your prefix ex. mollyc] + West for Territory Name and click Save.

  7. You will now have four Territories with your prefix.

Exercise 3 - Create Products

In this exercise, you will adding products and services for use on work orders and add them to the price list.

Task 1 - Add Inventory Product

  1. In the Dynamics 365 Field Service app, click the Settings area in the bottom-left of the sitemap, and in the General group select Products.

  2. Click Add Product.

  3. Enter [your prefix ex. mollyc] + Remote Printer for Name.

  4. Enter [your prefix ex. mollyc] + Print-Serv-1234 for Product ID.

  5. Select Default Unit for Unit Group.

  6. Select Primary Unit for Default Unit.

  7. Set Decimals Supported to 2.

  8. Select the Field Service tab.

  9. Set Field Service Product Type to Inventory.

  10. Set Taxable field to No.

  11. Click Save. (You may see a warning that a default price list has not been set. You can ignore this.)

  12. Click Publish and click Confirm from the popup.

  13. Click Save & Close

Task 2 - Add Non-Inventory Product

  1. Click Add Product.

  2. Enter [your prefix ex. mollyc] + Monthly Printer Maintenance for Name.

  3. Enter [your prefix ex. mollyc] + Print-Maint for Product ID.

  4. Select Default Unit for Unit Group.

  5. Select Primary Unit for Default Unit.

  6. Set Decimals Supported to 2.

  7. Select the Field Service tab.

  8. Set Field Service Product Type to Non-Inventory.

  9. Set Taxable field to No.

  10. Click Save. (You may see a warning that a default price list has not been set. You can ignore this.)

  11. Click Publish and click Confirm from the popup.

  12. Click Save & Close

Task 3 - Add Service Product

  1. Click Add Product.

  2. Enter [your prefix ex. mollyc] + Printer Service Fee for Name.

  3. Enter [your prefix ex. mollyc] + Printer-Service-Fee for Product ID.

  4. Select Default Unit for Unit Group.

  5. Select Primary Unit for Default Unit.

  6. Set Decimals Supported to 2.

  7. Select the Field Service tab.

  8. Set Field Service Product Type to Service.

  9. Set Taxable field to No.

  10. Click Save. (You may see a warning that a default price list has not been set. You can ignore this.)

  11. Click Publish and click Confirm from the popup.

  12. Click Save & Close

Task 4 - Add Products to a Price List

  1. In the Dynamics 365 Field Service app, click the Settings area in the bottom-left of the sitemap, and in the General group select Price Lists.

  2. Click + New.

  3. Enter [your prefix ex. mollyc] + Price List for Name.

  4. Click Save.

  5. Select the Price List Items tab.

  6. Click + New Price List Item.

  7. Select the [your prefix] Remote Printer product you created in Task 1.

  8. Select Primary Unit for Unit.

  9. Select the Pricing Information tab.

  10. Select Currency Amount from the Pricing Method drop-down field.

  11. Enter 1000.00 for Amount.

  12. Click Save & Close.

  13. Click + New Price List Item.

  14. Select the [your prefix] Monthly Printer Maintenance product you created in Task 2.

  15. Select Primary Unit for Unit.

  16. Select the Pricing Information tab.

  17. Select Currency Amount from the Pricing Method drop-down field.

  18. Enter 750.00 for Amount.

  19. Click Save & Close.

  20. Click + New Price List Item.

  21. Select the [your prefix] Printer Service Fee product you created in Task 3.

  22. Select Primary Unit for Unit.

  23. Select the Pricing Information tab.

  24. Select Currency Amount from the Pricing Method drop-down field.

  25. Enter 150.00 for Amount.

  26. Click Save & Close.

  27. Click Related on the Price List and select Field Service Price List Items.

  28. Click + New Field Service Price List Item.

  29. Enter [your prefix ex. mollyc] + Printer Service Fee for Name.

  30. Select Yes from the Flat Fee drop-down field.

  31. Select the [your prefix] Price List price list you created in Task 4.

  32. Select the [your prefix] Printer Service Fee product you created in Task 3.

  33. Click Save & Close.
