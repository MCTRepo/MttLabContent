Practice Lab 3 - Resources

Exercise 1 – Base location

In this exercise you will create the location where resources will start and end their day.

Task 1 - Set Time Zone

  1. Click the Settings icon in the navigation bar and select Personalize Settings.

  2. Select your Time Zone and click OK.

  3. Click on Home at the top of the left-hand side navigation.

  4. In the Dynamics 365 Field Service app, click the Service area in the bottom-left of the sitemap, and in the Scheduling group select Schedule Board.

  5. Click on the Scheduler Settings gear icon on the Schedule Board.

  6. Select your Time Zone

Task 2 - Create Organizational Unit

  1. In the Dynamics 365 Field Service app, click the Settings area in the bottom-left of the sitemap, and in the General group select Org Units.

  2. Click + New.

  3. Enter [your prefix ex. mollyc] + Org Unit for Name.

  4. Select the Scheduling tab.

  5. Enter 47.70939 for Latitude and -122.31784 for Longitude.

  6. Click Save & Close

Task 3 - Create a Warehouse for a Truck

  1. In the Dynamics 365 Field Service app, click the Inventory area in the bottom-left of the sitemap, and in the Inventory group select Warehouses.

  2. Click + New.

  3. Enter [your prefix ex. mollyc] + Truck for Name.

  4. Click Save & Close

Exercise 2 – Configure resources

Task 1 - Create a Bookable Resource for your user record

  1. In the Dynamics 365 Field Service app, click the Resources area in the bottom-left of the sitemap, and in the Resource group select Resources.

  2. Click + New.

  3. Select User from the Resource Type drop-down field.

  4. Select the user record you are signed in as ex. mollyc or Molly Clark for the User.

  5. Select your Time Zone for Time Zone.

  6. Select the Scheduling tab.

  7. Select Organizational Unit Address from the Start Location drop-down field.

  8. Select Organizational Unit Address from the End Location drop-down field.

  9. Select the [your prefix] Org Unit organizational unit you created in Exercise 1 for Organizational Unit.

  10. Select the Field Service tab

  11. Set the Hourly Rate field to 175.

  12. Select the [your prefix] Truck you created in Exercise 1 for Warehouse.

  13. Select Yes from the Time Off Approval Required drop-down field.

  14. Click Save.

  15. Select the General tab

  16. If Resource Characteristics are not shown in a sub-gird on the General tab, click Related and select *Resource Characteristics**.

  17. Click + New Bookable Resource Characteristic.

  18. Select the [your prefix] Building Security you created in the previous lab for Characteristic.

  19. Select the [your prefix] Level 5 Security you created in the previous lab for Rating Value.

  20. Click Save and Close.

  21. Click + New Bookable Resource Characteristic.

  22. Select the [your prefix] CISM you created in the previous lab for Characteristic.

  23. Select the [your prefix] Proficient you created in the previous lab for Rating Value.

  24. Click Save and Close.

  25. Click + New Bookable Resource Characteristic.

  26. Select the [your prefix] CISSP you created in the previous lab for Characteristic.

  27. Select the [your prefix] Expert you created in the previous lab for Rating Value.

  28. Click Save and Close.

  29. Select the General tab

  30. If Resource Categories are not shown in a sub-gird on the General tab, click Related and select *Resource Characteristics**.

  31. Click + New Bookable Resource Category Assn.

  32. Select the [your prefix] Installation Specialist you created in the previous lab for Resource Category.

  33. Click Save & Close.

  34. Click + New Bookable Resource Category Assn.

  35. Select the [your prefix] Security Analyst you created in the previous lab for Resource Category.

  36. Click Save and Close.

  37. Click Related and select Resource Territories.

  38. Click + New Resource Territory.

  39. Select the [your prefix] North record you created in the previous lab for Territory.

  40. Click Save & Close.

  41. Click Show Work Hours in the command bar.

  42. In the Calendar view, click + New drop-down arrow and select Working hours.

  43. Select the first of the current month for the date.

  44. Select Every Week from the Repeat drop-down field.

  45. Check Mo thru Fr are checked

  46. Uncheck Su and Sa.

  47. Set the Work Hour are set to 9:00 AM to 5:00 PM

  48. Select your Time Zone.

  49. Click Save.

  50. Click Save & Close.

Task 2 - Create a Working Hours template

  1. In the Dynamics 365 Field Service app, click the Resources area in the bottom-left of the sitemap, and in the Resource group select Work Hours Templates.

  2. Click + New.

  3. Enter [your prefix ex. mollyc] + Standard hours for Name.

  4. Select your user for Template resource.

  5. Click Save & Close.
