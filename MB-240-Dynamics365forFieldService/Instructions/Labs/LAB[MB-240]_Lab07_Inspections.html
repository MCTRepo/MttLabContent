Practice Lab 7 - Inspections

Exercise 1 - Create and use an Inspection Template

In this exercise you will be defining an inspection template and adding it to a server type task.

Task 1 - Create Inspection Template

  1. In the Dynamics 365 Field Service app, click the Settings area in the bottom-left of the sitemap, and in the Work Orders group select Inspection Templates.

  2. Click + New.

  3. Enter [your prefix ex. mollyc] + Printer Inspection for Name for Inspection.

  4. Click on TextBox to add question.

  5. Enter Serial number for Question1 title.

  6. Toggle Required to On.

  7. Click Advanced.

  8. Select Barcode for Input type.

  9. Select Toolbox tab.

  10. Click on Dropdown to add question.

  11. Enter Condition for Question2 title.

  12. Toggle Required to On.

  13. Change Item 1 to Poor.

  14. Change Item 2 to Satisfactory.

  15. Change Item 3 to Good.

  16. Click on Number to add question.

  17. Enter Page Count for Question3 title.

  18. Select the Advanced tab.

  19. Toggle Whole Number to On.

  20. Select the Toolbox tab.

  21. Click on TextBox to add question.

  22. Enter Comments for Question4 title.

  23. Select the Advanced tab.

  24. Select Multiline for the Input type dro-pdown.

  25. Select the Toolbox tab.

  26. Click on File to add question.

  27. Enter Image for Question5 title.

  28. Click the Preview tab.

  29. Click Save.

  30. Click Publish and click Publish again.

Task 2 - Add Inspection Template to Service Type Task

  1. In the Dynamics 365 Field Service app, click the Settings area in the bottom-left of the sitemap, and in the Work Orders group select Service Task Types.

  2. Click + New.

  3. Enter [your prefix ex. mollyc] + Inspect Printer for Name.

  4. Enter 5 Minutes for Estimated Duration.

  5. Toggle Has Inspection to Yes.

  6. Select the [your prefix] Printer Inspection inspection template you created in Task 1 for Inspection Template.

  7. Click Save.

  8. In the Dynamics 365 Field Service app, click the Settings area in the bottom-left of the sitemap, and in the Work Orders group select Incident Types.

  9. Edit the [your prefix] Service Printer incident type.

  10. Select the Service Tasks tab.

  11. Click + New Incident Type Service Task.

  12. Enter [your prefix ex. mollyc] + Inspect Printer for Name.

  13. Select the [your prefix] Inspect Printer service task type you created in Task 2 for Task Type.

  14. Click Save and Close.

  15. Click Save & Close.

Task 3 – Download an image

  1. On a browser, navigate to https://www.microsoft.com/en-us/microsoft-365/blog/2019/07/09/lexmark-digitizes-its-future-with-modern-workplace-microsoft-365/.

  2. Right click on the image of the printer and select Save image as and click OK.

Task 4 – Create a new Work Order and view the inspection

  1. In the Dynamics 365 Field Service app, click the Service area in the bottom-left of the sitemap, and in the Scheduling group select Work Orders.

  2. Click + New.

  3. Select the [your prefix] Relecloud account you created in Task 1 for Service Account.

  4. Select the [your prefix] Service Printer incident type you created in a previous lab for Primary Incident Type.

  5. Click Save.

  6. Wait about 30 seconds to a minute and click Refresh in the command bar.

  7. Select the Service Tasks tab and verify that the four tasks were added.

  8. Open the [your prefix] Inspect Printer work order service task.

  9. Enter 122333 for Serial number.

  10. Select Satisfactory from the Condition drop-down field.

  11. Enter 999 for Page Count.

  12. Enter No problems found for Comments.

  13. Click on Choose file(s) and browse to the image you downloaded in Task 3 and click Open.

  14. Click Save.

  15. Select Pass for Result.

  16. Click Mark Complete.
