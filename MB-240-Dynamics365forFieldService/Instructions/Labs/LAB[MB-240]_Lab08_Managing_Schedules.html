Practice Lab 8 - Scheduling

Exercise 1 - Schedule work orders with the schedule board

Out-of-the-box, Dynamics 365 for Field Service has the work order table enabled for use with the Resource Scheduling feature. In this exercise, we will be creating work orders that we can schedule using the application.

Task 1 – Create work orders

  1. In the Dynamics 365 Field Service app, click the Service area in the bottom-left of the sitemap, and in the Scheduling group select Work Orders.

  2. Click + New.

  3. Select the [your prefix] Relecloud account you created in an earlier lab for Service Account.

  4. Select the [your prefix] Service Printer incident type you created in a previous lab for Primary Incident Type.

  5. Select the Settings tab.

  6. Select the [your prefix] Normal priority you created in a previous lab for Priority.

  7. Select the [your prefix] North territory you created in a previous lab for Service Territory.

  8. Enter Today \@ 1:00 PM for Time from Promised.

  9. Enter Tomorrow \@ 3:00 PM for Time to Promised.

  10. Click Save. Make a note of the Work Order number.

  11. Click + New.

  12. Select the [your prefix] Relecloud account you created in an earlier lab for Service Account.

  13. Select the [your prefix] Quick Inspection incident type you created in a previous lab for Primary Incident Type.

  14. Select the Settings tab.

  15. Select the [your prefix] Urgent priority you created in a previous lab for Priority.

  16. Select the [your prefix] North territory you created in a previous lab for Service Territory.

  17. Click Save. Make a note of the Work Order number.

Exercise 2 - Schedule work orders using the schedule board

Universal Resource Scheduling provides several items that can be used to assist in scheduling resources for specific items. The two primary components that are used are the Schedule Board and the Schedule Assistant. The Schedule Board provides the ability to manually schedule items, and the assistant offers suggestions on resources based on factors like location, skills, and availability. In this exercise we will examine how you can use the schedule board to schedule items.

Task 1 – Enable the new Schedule Board

  1. In the Dynamics 365 Field Service app, click the Service area in the bottom-left of the sitemap, and in the Scheduling group select Schedule Board.

  2. In the top-right on the Schedule Board, toggle New Schedule Board to On.

Task 2 – Schedule work orders

  1. In the Dynamics 365 Field Service app, click the Service area in the bottom-left of the sitemap, and in the Scheduling group select Schedule Board.

  2. The Schedule Board provides several options that can be used to schedule items, such as a filter, and a map view.

  3. Expand the Booking Requirements pane at the bottom of the board.

  4. Select the Unscheduled Work Orders tab.

  5. Select the Urgent work order that you created in Exercise 1 and drag it to an available time slot for your user’s resource on the schedule board.

  6. Release the mouse button and the item will be placed on the schedule board.

  7. Select the Normal work order that you created in Exercise 1 and click Find Availability.

  8. Dynamics 365 will analyze the requirements needed for this item and will factor in other items such as any skills required, work order & resource locations, and resource availability to create a list of suggested resources that would be able to work on this item.

  9. Click Exit Schedule Assistant.

  10. Click on the Filters icon.

  11. Select the [your prefix] North territory you created in a previous lab.

  12. Click Apply. Your resource should be listed.

  13. Right-click on your resource cell in the Schedule Board and select View resource card.

  14. Click on the Map View icon.

  15. Select the Normal work order that you created in Exercise 1 and zoom in on the map.

  16. Click on View Settings in the map and enable Traffic.

  17. Click on the Map View icon to close the map pane.

  18. Close the Filters pane.

  19. Click on the Legend icon and examine the colors used on the board.

  20. Click on the booking for the work order you dragged onto the schedule board earlier in this task. The tooltip for the booking will be displayed.

  21. Click on the Detail panel icon. The details of the booking will be displayed in the details panel.

  22. Select the Normal work order that you created in Exercise 1 and click Find Availability.

  23. Select your resource and click Book & Exit.
