Practice Lab 13 – Power Platform

Scenario

Dispatchers have requested more information about the customer and work order to be displayed on the schedule board. You are going to customize the views used by the schedule board to achieve this.

Exercise 1: Gather components to customizes

In this exercise, you will create a solution and add the views used on the schedule board/

Task 1: Create solution

  1. Navigate to the Power Apps Maker portal https://make.powerapps.com

  2. Select the WWLLABnnn environment.

  3. Click Solutions.

  4. Click + New solution.

  5. Enter [your prefix ex. mollyc] + Solution for Name.

  6. Click + New publisher.

  7. Enter [your prefix ex. mollyc] for Display Name.

  8. Enter [your prefix ex. mollyc] for Name.

  9. Enter [your prefix ex. mollyc] for Prefix.

  10. Click Save.

  11. Select the [your prefix] publisher.

  12. Click Create.

Task 2: Add views to solution

  1. Click on [your prefix ex. mollyc] Solution.

  2. Click Add existing and select Table.

  3. Select Bookable Resource Booking and click Next

  4. Click Select components.

  5. Select the Views tab.

  6. Select Schedule Board Details Pane Field Layout.

  7. Select Work Order Booking Map Tooltips view.

  8. Click Add.

  9. Click Add.

  10. Click Add existing and select Table.

  11. Select Resource Requirement and click Next

  12. Click Select components.

  13. Select the Views tab.

  14. Select Active Resource Requirements.

  15. Select Unscheduled Work Order Requirements.

  16. Click Add.

  17. Click Add.

  18. Click Add existing and select Table.

  19. Select Bookable Resource and click Next.

  20. Click Select components.

  21. Select the Views tab.

  22. Select Active Bookable Resources.

  23. Click Add.

  24. Click Add.

Task 3: Customize views

  1. Edit your [your prefix ex. mollyc] Solution.

  2. Select Bookable Resource table.

  3. Edit the Active Bookable Resources view.

  4. Add the Start Location column to the view.

  5. Add the Organizational Unit column to the view.

  6. Add the Hourly Rate column to the view.

  7. Click Save As.

  8. Enter [your prefix ex. mollyc] + Bookable Resource Detail panel for Name and click Save.

  9. Click Publish.

  10. Click Back.

  11. Select Bookable Resource Booking table.

  12. Select the Views tab.

  13. Click + Add view.

  14. Enter [your prefix ex. mollyc] + BRB Details panel for Name and click Create.

  15. Add the Work Order column to the view.

  16. Add the Booking Status column to the view.

  17. Add the Duration column to the view.

  18. Add the Resource column to the view.

  19. Select the Related tab and expand Work Order.

  20. Add the Work Order Type column to the view.

  21. Add the Primary Incident Customer Asset column to the view.

  22. Click Save.

  23. Click Publish.

  24. Click Back.

  25. Select Bookable Resource Booking table.

  26. Select the Views tab.

  27. Click + Add view.

  28. Enter [your prefix ex. mollyc] + BRB Tooltip for Name and click Create.

  29. Add the Work Order column to the view.

  30. Remove the Name column.

  31. Add the Resource column to the view.

  32. Add the Booking Status column to the view.

  33. Select the Related tab and expand Work Order.

  38. Click Save.

  41. Select Resource Requirement table.

  42. Select the Views tab.

  43. Click + Add view.

  44. Enter [your prefix ex. mollyc] + Unscheduled North Territory Work Orders for Name and click Create.

  45. Add the Work Order column to the view.

  46. Remove the Name column.

  48. Add the Priority column to the view.

  55. Click the arrow to sort descending.

  58. Select Remaining Duration with operator Is greater than and value 0 minutes.

  63. Click Save.

  65. Click Back.

Task 4: Use views on schedule board

  1. In the Dynamics 365 Field Service app, click the Service area in the bottom-left of the sitemap, and in the Scheduling group select Schedule Board.

  2. Select the [your prefix] Board tab.

  3. Click the ellipsis (…) next to the name of the tab and select Board settings.

  4. Select the Map tab.

  5. Select [your prefix] Bookable Resource Detail panel for Resource details view.

  6. Expand Schedule types and select Other.

  7. Select [your prefix] BRB Details panel for Booking details view.

  8. Expand Schedule types and select Work Order.

  9. Select [your prefix] BRB Tooltip for Booking tooltips view.

  10. Select [your prefix] BRB Details panel for Booking details view.

  11. Select Requirement panels.

  12. Toggle Show default requirement panels to Off.

  13. Click on +.

  14. Enter North work orders to schedule for Title.

  15. Select [your prefix] Unscheduled North Territory Work Orders for View.

  16. Click Save.

Task 5: Test the changes

  1. Verify that the requirements panel only shows the view you created.

  2. Select the a booking and verify the tooltip reflects the changes you made.

  3. Click on the Detail panel icon.

  4. Click on the resource cell and verify the details pane reflects the changes you made.
